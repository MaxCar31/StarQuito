¿Qué es START?

Impulsando el espíritu empresarial

Fundada en 1996, START Global es la iniciativa líder en el fomento de la próxima generación de emprendedores para cambiar nuestro mundo a través del espíritu empresarial, la tecnología y la innovación. Están impulsados por la Universidad de San Galo (HSG), una de las principales universidades empresariales de Europa.

Conocer Start Global

Acerca de Start Quito

Conectando el emprendimiento

START Quito es una organización independiente estudiantil que forma parte de la red de START Network de la organización estudiantil START Global ubicada en San Galo, Suiza.
